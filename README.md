# CodingFactory9 - Web MVC DB First Project

## 📌 Description

This project is a web application developed using ASP.NET MVC architecture with a Database First approach.
It was created as part of the Coding Factory program and demonstrates full-stack development concepts including database integration, MVC structure, and CRUD operations.

---

## 🛠️ Technologies Used

* ASP.NET MVC
* Entity Framework (Database First)
* SQL Server
* C#
* HTML / CSS / JavaScript

---

## ⚙️ Features

* Create, Read, Update, Delete (CRUD) operations
* Database integration using Entity Framework
* MVC architecture (Model - View - Controller)
* Structured and maintainable codebase

---

## 🚀 Getting Started

### Prerequisites

* Visual Studio
* SQL Server
* .NET Framework

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/CodingFactory9-WebMVCDBFirstPro.git
   ```

2. Open the project in Visual Studio

3. Configure the database connection:

   * Update the `connection string` in `Web.config`

4. Build and run the project

---

## 🗄️ Database

This project follows the **Database First** approach:

* The database is created first in SQL Server
* Entity Framework models are generated from the database
